# Bienestar Integral PWA

Una aplicación web progresiva diseñada para mejorar tu salud física y mental de forma personalizada.

## Características
- **Onboarding Inteligente**: 20 preguntas con lógica condicional para entender tus necesidades.
- **Planes Personalizados**: Rutinas de bajo impacto (hernias, lesiones), nutrición simple y protocolos de sueño.
- **Dashboard de Seguimiento**: Visualiza tus objetivos y progreso diario.
- **PWA e Interacción Offline**: Instalable en móvil y escritorio, funcional sin conexión a internet.
- **Despliegue Automático**: Configurado con GitHub Actions para GitHub Pages.

## Instalación
1. Sube este repositorio a GitHub.
2. Habilita GitHub Pages en la configuración del repositorio (`Settings > Pages`).
3. La aplicación se desplegará automáticamente.

## Stack Tecnológico
- HTML5 & CSS3 (Tailwind CSS)
- JavaScript Vanilla
- Service Workers (PWA)
- GitHub Actions (CI/CD)
